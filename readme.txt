1. Install package:

Install Node.js

npm install

npm install axios bootstrap react-router-dom bootstrap-icons

npm install mysql cors express nodemon


2. Import mysql database (libraryDB) and run mysql server (e.g. xampp)

3. go to project folder => cd server => run npm start

4. go to project folder => cd librarySystem => npm run dev

5. open "localhost:5173/adminlogin" in browser to run the app
